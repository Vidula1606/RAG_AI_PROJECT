## Overview

RAG_AI_PROJECT is an open-source developer toolkit that facilitates knowledge ingestion, semantic search, and visualization within insurance-focused AI systems. It enables efficient document management and intelligent, context-aware interactions, streamlining the development of sophisticated insurance applications.

**Why RAG_AI_PROJECT?**

This project aims to simplify the integration of knowledge bases and conversational AI in insurance tech. The core features include:

- 🧊 **🧬 Embedding & Segmentation:** Ingests and embeds documents into a local vector database for fast retrieval.
- 🎨 **🖼️ 3D Visualization:** Provides visual insights into document embeddings to understand their relationships.
- 🧊 **🔍 Hybrid Search:** Combines BM25 and semantic search for accurate, context-aware responses.
- 🧊 **🛠️ Database Inspection:** Offers quick health checks and data overview of the vector store.
- 🎯 **🌐 Web Interface:** Interactive UI for querying and managing insurance knowledge bases.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Python
- **Package Manager:** Conda

### Installation

Build RAG_AI_PROJECT from the source and install dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/watash1desu/RAG_AI_PROJECT
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd RAG_AI_PROJECT
    ```

3. **Install the dependencies:**

**Using [conda](https://docs.conda.io/):**

```sh
❯ conda env create -f conda.yml
```

### Usage

Run the project with:

**Using [conda](https://docs.conda.io/):**

```sh
conda activate {venv}
python {entrypoint}
```

### Testing

Rag_ai_project uses the {__test_framework__} test framework. Run the test suite with:

**Using [conda](https://docs.conda.io/):**

```sh
conda activate {venv}
pytest
```
